# Spot-On — Architecture Diagrams

This directory contains the visual documentation and architecture diagrams for the Spot-On project.

## Structure

- `class/`: Contains class diagrams (e.g., Domain concepts, database models).
- `component/`: Contains component and architecture diagrams explaining the backend/frontend structure and service relationships.

These diagrams help maintain a high-level overview of the system and are useful for onboarding new developers.
